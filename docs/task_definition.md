# 任务定义

## 研究目标

构建面向交通管理场景的多模态风险研判模型。模型输入交通监控图像或视频关键帧，以及天气、交通流、平均速度、外部事件等结构化信息，输出当前路段风险等级、异常事件类型、原因解释和管理建议。

## 输入

- 图像：D2-City 视频抽帧或原始图片。
- 结构化上下文：`weather`、`traffic_flow`、`average_speed`、`event_info`。
- 用户问题：要求模型结合画面与结构化信息完成风险研判。

## 输出

- 风险等级：高风险、中风险、低风险。
- 异常事件：事故相关拥堵、施工影响、活动散场拥堵、车辆故障、交通拥堵、正常通行。
- 原因分析：结合视觉证据和外部上下文说明判断依据。
- 管理建议：给出巡检、诱导、处置或常规观察建议。

## 当前阶段

D2-City 没有天然 QA 标注，当前项目先用规则生成弱标注 SFT 样本，用于验证数据链路和多模态 LoRA 微调可行性。后续论文实验可引入人工复核标签、事件检测模型输出、轨迹/速度估计结果和真实管理记录。
